Bitworse AA ヽ(´ω｀)ノ v0 (2026)
=============================
![example photo for Bitworse AA.](/samples/sample1.png)  
A simple web tool to let you make AA based on 2channel and Ayashii World.  
Probably worse than the actual programs... but it works.  

Current Features:
- Collection of symbols to build your AA with
- Templates of characters from Ayashii World and 2Channel
- Button that lets you save your AA to disk without you having to do it yourself
- That one bug from Mattari-AA is fixed
- Uh...

Usage
------
You can either use [The copy hosted on Nekoweb](https://bloxy97.nekoweb.org/aa/) or run it from your own system. Here's how.

1. Download the source code
2. Unzip the source code somewhere in a suitable habitat on your disk (make sure not to change anything inside the root folder)
3. Open index.html in your web browser of choice
4. profit
