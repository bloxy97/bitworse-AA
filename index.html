Information
  • You can use this page to design lots of ASCII Art (AA) from Ayashii World and 2channel.
  • Ayashii World usually uses MS Gothic and 2channel uses MS PGothic. IPAMonaGothic and IPAMonaPGothic can be used instead, though.
  • MS Gothic (Ayashii) AA will probably look odd using the MS PGothic (2ch) font and vice versa. This can usually be fixed manually.
  • JavaScript is required to use the buttons and style options.
  • Some buttons will provide more information if you hover over them.
  • If you need a place to post your AA, try these guys, maybe. (ВҙПүпҪҖ)
